Hello World

  1. Crear el repositorio en github
  2. Crear la carpeta donde se almacenará nuestro proyecto
  3. Abrir la carpeta con Visual Studio Code
  4. Abrir la terminal integrada (ctrl + ñ)
  5. git init (iniciará nuestro proyecto como git)
  6. git remote add origin + url del repositorio (indicará origen del repositorio)
  7. git add + nombre de archivo (agregará cambios al repositorio, por ejemplo: git add index.html)
  8. git commit -m "nombredelcommit" (otorgará nomre a la piscina de cambios)
  9. git push -u origin master (crea la rama dentro del repositorio y sube el código
  10. Finalmente, cuando seguimos haciendo cambios ejecutamos git push (luego de haber agregado los cambios y realizado commit)
